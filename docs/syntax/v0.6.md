# NOVA Language Specification

### Version 0.6

NOVA v0.6 expands the language by introducing iteration through while loops, range loops, array iteration, loop scope, and loop control statements.

---

# New Features

## While Loops

Execute a block repeatedly while a condition evaluates to `true`.

```text
i: N = 0

while i < 5 {

    print(i)

    i = i + 1

}
```

Conditions must evaluate to a boolean value.

```text
while isRunning {

}
```

```text
while age < 18 {

}
```

Invalid:

```text
while 10 {

}
```

Produces:

```text
Condition must evaluate to a Boolean value.
```

---

## Range Loops

Iterate over a numeric range.

Exclusive range:

```text
for i in 0..5 {

    print(i)

}
```

Inclusive range:

```text
for i in 0...5 {

    print(i)

}
```

Ranges automatically descend when the starting value is greater than the ending value.

```text
for i in 5..0 {

    print(i)

}
```

---

## Iterable Types

The following values are iterable:

* Arrays
* Nested Arrays
* Ranges

---

## Array Iteration

Arrays may be traversed directly.

```text
names: [S] = [
    "Varun",
    "Alex",
    "John"
]

for name in names {

    print(name)

}
```

---

## Nested Loops

Loops may be nested arbitrarily.

```text
for i in 0..3 {

    for j in 0..3 {

        print(i, j)

    }

}
```

---

## Break Statement

`break` immediately exits the nearest enclosing loop.

```text
for i in 0...10 {

    if i == 5 {

        break

    }

}
```

`break` may only be used inside loops.

Invalid:

```text
break
```

Produces:

```text
Break statement can only be used inside a loop.
```

---

## Continue Statement

`continue` skips the current iteration of the nearest enclosing loop.

```text
for i in 0...5 {

    if i == 2 {

        continue

    }

    print(i)

}
```

`continue` may only be used inside loops.

Invalid:

```text
continue
```

Produces:

```text
Continue statement can only be used inside a loop.
```

---

## Loop Scope

Variables declared inside a loop exist only within that loop.

```text
while true {

    temp: N = 10

}
```

```text
print(temp)
```

Produces:

```text
Variable 'temp' is not declared.
```

---

## Loop Variables

Loop variables are automatically declared by the runtime.

```text
for i in 0..5 {

    print(i)

}
```

They exist only inside the loop body.

```text
print(i)
```

Produces:

```text
Variable 'i' is not declared.
```
