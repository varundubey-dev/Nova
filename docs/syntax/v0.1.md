# NOVA Language Specification

### Version 0.1

# Features

NOVA v0.1 supports:

* Numbers
* Strings
* Variables
* Arithmetic Expressions
* Output
* Comments

Everything else is reserved for future versions.

---

# Primitive Types

| Type | Description |
| ---- | ----------- |
| `S`  | String      |
| `N`  | Number      |

Examples:

```text
name: S = "Varun"

age: N = 19

height: N = 175.5
```

---

# Variable Declaration

Variables are mutable by default.

### Declaration With Assignment

```text
name: S = "Varun"

age: N = 19
```

### Declaration Without Assignment

```text
name: S

age: N
```

### Assignment Later

```text
name = "Varun"

age = 19
```

---

## Uninitialized Variables

Accessing an uninitialized variable results in a runtime error.

```text
name: S

print(name)
```

Error:

```text
Variable 'name' accessed before initialization.
```

---

# Assignment

Variables may be reassigned.

```text
age: N = 19

age = 20
```

The assigned value must match the declared datatype.

```text
name: S = "Varun"

name = 100
```

Error:

```text
Datatype mismatch.
```

---

# Arithmetic Operators

| Operator | Purpose        |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |

Examples:

```text
5 + 3

10 - 2

4 * 5

20 / 4
```

---

## Division Behavior

Division always produces a numeric result.

```text
5 / 2
```

Result:

```text
2.5
```

---

# Output

Output is performed using `print()`.

```text
print("Hello World")
```

```text
print(5 + 3)
```

```text
print(name)
```

---

# Comments

## Single-Line Comments

```text
// This is a comment
```

## Multi-Line Comments

```text
/ *
This is a comment
spanning multiple lines
* /
```

---

# Program Execution

NOVA executes statements from top to bottom.

Example:

```text
name: S = "Varun"

age: N = 19

print(name)

print(age + 1)
```
