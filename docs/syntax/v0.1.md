# NOVA Language Specification

### Version 0.1

NOVA v0.1 defines the core syntax and execution model of the language.

This version introduces NOVA's first primitive types, variable declarations, arithmetic expressions, output, and comments.

---

# Features

NOVA v0.1 supports:

* Numbers
* Strings
* Variables
* Arithmetic Expressions
* Output
* Comments

Everything else is reserved for future versions.

---

# Primitive Types

| Type | Description |
| ---- | ----------- |
| `S`  | String      |
| `N`  | Number      |

Examples:

```text
name: S = "Varun"

age: N = 19

height: N = 175.5
```

---

# Variable Declaration

Variables are mutable by default.

### Declaration With Assignment

```text
name: S = "Varun"

age: N = 19
```

### Declaration Without Assignment

```text
name: S

age: N
```

### Assignment Later

```text
name = "Varun"

age = 19
```

---

# Uninitialized Variables

Variables may be declared without an initial value.

Accessing an uninitialized variable produces a runtime error.

```text
name: S

print(name)
```

Produces:

```text
Variable 'name' accessed before initialization.
```

---

# Assignment

Variables may be reassigned.

```text
age: N = 19

age = 20
```

Assigned values must match the declared datatype.

```text
name: S = "Varun"

name = 100
```

Produces:

```text
Datatype mismatch.
```

---

# Arithmetic Operators

| Operator | Purpose        |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |

Example:

```text
5 + 3

10 - 2

4 * 5

20 / 4
```

---

# Division

Division always produces a numeric result.

```text
5 / 2
```

Result:

```text
2.5
```

---

# Output

Output is performed using `print()`.

```text
print("Hello World")
```

```text
print(5 + 3)
```

```text
print(name)
```

---

# Comments

## Single-Line Comments

```text
// This is a comment
```

## Multi-Line Comments

```text
/*
This is a comment
spanning multiple lines
*/
```

---

# Program Execution

Programs execute sequentially from top to bottom.

```text
name: S = "Varun"

age: N = 19

print(name)

print(age + 1)
```
