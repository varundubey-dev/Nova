# NOVA Language Specification

### Version 0.2

NOVA v0.2 expands the language by introducing booleans, the any type, constants, null values, comparison operators, and logical operators.

These additions establish NOVA's first complete primitive type system and enable logical and comparison-based expressions.

---

# Features

NOVA v0.2 supports:

* Numbers
* Strings
* Booleans
* Any Type
* Variables
* Constants
* Arithmetic Expressions
* Comparison Expressions
* Logical Expressions
* Null Values
* Output
* Comments

---

# Primitive Types

| Type | Description |
| ---- | ----------- |
| `S`  | String      |
| `N`  | Number      |
| `B`  | Boolean     |
| `U`  | Any Type    |

Example:

```text
name: S = "Varun"

age: N = 19

isAdult: B = true

value: U = "Hello"
```

---

# Any Type

Variables declared as `U` may store values of any datatype.

```text
value: U = "Hello"

value = 100

value = false

value = null
```

---

# Null Values

`null` represents the absence of a value.

```text
name: S = null

age: N = null

value: U = null
```

Null values may be compared for equality.

```text
value == null
```

Operations other than equality are invalid.

```text
null + 5
```

Produces:

```text
Cannot perform arithmetic on null.
```

---

# Constants

Constants are declared using `::`.

```text
PI:: N = 3.14159
```

Constants may also be initialized later.

```text
VERSION:: S

VERSION = "1.0"
```

Once initialized, constants cannot be reassigned.

```text
VERSION = "2.0"
```

Produces:

```text
Cannot reassign immutable variable.
```

# Arithmetic Operators

| Operator | Purpose        |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |
| `%`      | Modulus        |

Example:

```text
10 % 3
```

---

# Comparison Operators

| Operator |
| -------- |
| `==`     |
| `!=`     |
| `<`      |
| `>`      |
| `<=`     |
| `>=`     |

Example:

```text
age >= 18
```

Produces:

```text
true
```

---

# Equality

Values of the same datatype are compared by value.

```text
5 == 5
```

Produces:

```text
true
```

Values of different datatypes are never equal.

```text
5 == "5"
```

Produces:

```text
false
```

---

# Logical Operators

| Operator | Meaning | 
| -------- | ------- |
| `&&`     | AND     | 
| `||`     | OR      |
| `!`      | NOT     |  

Example:

```text
isAdult && isStudent

loggedIn || isAdmin

!blocked
```

---

# Output

Output continues to use `print()`.

```text
print(age)

print(age >= 18)
```

---

# Comments

## Single-Line Comments

```text
// This is a comment
```

## Multi-Line Comments

```text
/*
This is a comment
spanning multiple lines
*/
```

---

# Program Execution

Programs execute sequentially from top to bottom.

```text
name: S = "Varun"

age: N = 19

isAdult: B = age >= 18

print(isAdult)
```
