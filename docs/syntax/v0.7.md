# NOVA Language Specification

### Version 0.7

NOVA v0.7 expands the language by introducing user-defined functions, parameters, return values, recursion, and function-local scope.

---

# New Features

## Function Declarations

Functions are declared using the `fn` keyword.

```text
fn greet() {

    print("Hello")

}
```

Function declarations are resolved before program execution begins, allowing functions to be called before their declaration.

---

## Function Calls

Functions are invoked using their name followed by parentheses.

```text
greet()
```

Arguments are evaluated before being passed to the function.

Function calls may appear anywhere an expression is allowed.

```text
result: N = (
    double(10) +
    double(5)
)
```

Nested function calls are also supported.

```text
print(
    add(
        add(1, 2),
        add(3, 4)
    )
)
```

---

## Parameters

Functions may declare zero or more typed parameters.

```text
fn greet(name: S) {

    print(name)

}
```

```text
fn add(
    a: N,
    b: N
) -> N {

    return a + b

}
```

Arguments must match the declared parameter datatypes.

Invalid:

```text
greet(10)
```

Produces:

```text
Datatype mismatch.
```

---

## Return Values

Functions may return values using `return`.

```text
fn square(value: N) -> N {

    return value * value

}
```

Functions without a declared return type may omit `return` or return any value.

```text
fn greet() {

    print("Hello")

}
```

---

## Return Type Enforcement

Functions declaring a return datatype must terminate by returning a compatible value or `null`.

```text
fn add(a: N, b: N) -> N {

    return a + b

}
```

Invalid:

```text
fn add(a: N, b: N) -> N {

}
```

Produces:

```text
Function 'add' must return a value.
```

---

## Early Returns

`return` immediately terminates function execution.

```text
fn test() {

    print("Start")

    return

    print("Never Runs")

}
```

---

## Function Scope

Each function executes inside its own runtime scope.

Variables declared inside a function exist only within that function.

```text
fn test() {

    temp: N = 10

}
```

```text
print(temp)
```

Produces:

```text
Variable 'temp' is not declared.
```

Functions automatically inherit access to variables declared in parent scopes.

```text
name: S = "Nova"

fn greet() {

    print(name)

}
```

Local variables may shadow variables from outer scopes.

```text
name: S = "Global"

fn test() {

    name: S = "Local"

    print(name)

}
```

---

## Recursive Functions

Functions may invoke themselves recursively.

```text
fn factorial(n: N) -> N {

    if n <= 1 {

        return 1

    }

    return n * factorial(
        n - 1
    )

}
```

Recursive functions should eventually reach a terminating condition.
