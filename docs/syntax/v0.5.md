# NOVA Language Specification

### Version 0.5

NOVA v0.5 expands the language by introducing conditional execution, block scope, variable shadowing, and ternary expressions.

These features allow programs to execute different code paths based on runtime conditions while introducing lexical block scoping.

---

# New Features

## If Statements

Execute a block when a condition evaluates to `true`.

```text
age: N = 20

if age >= 18 {
    print("Adult")
}
```

Conditions must evaluate to a boolean value.

```text
if isAdmin {

}
```

```text
if age >= 18 && citizen {

}
```

Invalid:

```text
if 10 {

}
```

Produces:

```text
Condition must evaluate to a Boolean value.
```

---

## Else Statements

Provide an alternative execution path.

```text
if age >= 18 {

    print("Adult")

}
else {

    print("Minor")

}
```

---

## Else If Statements

Multiple conditions may be chained.

```text
if score >= 90 {

    print("A")

}
else if score >= 80 {

    print("B")

}
else {

    print("C")

}
```

---

## Nested Conditionals

Conditional blocks may be nested.

```text
if age >= 18 {

    if citizen {

        print("Eligible")

    }

}
```

---

## Ternary Expressions

Select between two values using a conditional expression.

```text
condition ? value_if_true : value_if_false
```

Example:

```text
status: S = (
    age >= 18
        ? "Adult"
        : "Minor"
)
```

Ternary expressions may appear anywhere an expression is allowed.

```text
print(
    isAdmin
        ? "Admin"
        : "User"
)
```

```text
largest: N = (
    x > y
        ? x
        : y
)
```

Both branches must produce compatible values.

Invalid:

```text
value: N = (
    false
        ? 10
        : "Hello"
)
```

Produces:

```text
Datatype mismatch.
```

---

## Parenthesized Expressions

Parentheses may be used to group expressions.

```text
value: N = (
    10 + 20
) * 3
```

They may also improve readability for multiline expressions.

```text
result: S = (
    age >= 18
        ? "Adult"
        : "Minor"
)
```

---

## Block Scope

Variables declared inside a block exist only within that block.

```text
if true {

    message: S = "Hello"

}
```

```text
print(message)
```

Produces:

```text
Variable 'message' is not declared.
```

Variables declared outside the block remain accessible.

```text
name: S = "Varun"

if true {

    print(name)

}
```

---

## Variable Shadowing

Inner scopes may shadow variables from outer scopes.

```text
name: S = "Global"

if true {

    name: S = "Local"

    print(name)

}

print(name)
```

---

## Output

Output is performed using `print()`.

`print()` accepts one or more comma-separated expressions.

```text
print("Hello")
```

```text
print(name)
```

```text
print(name, age)
```

```text
print("Score:", score)
```

Output values are separated by a single space.

Example:

```text
name: S = "Nova"

age: N = 1

print(name, age)
```

Produces:

```text
Nova 1
```
