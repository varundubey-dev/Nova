# NOVA Language Specification

### Version 0.5

NOVA v0.5 expands upon v0.4 by introducing:

* Conditional Execution
* If Statements
* Else Statements
* Else If Statements
* Ternary Operator
* Block Scope
* Parenthesized Multiline Expressions

---

# New Features

## If Statement

Execute a block only when a condition evaluates to `true`.

```text
age: N = 20

if age >= 18 {
    print("Adult")
}
```

Output:

```text
Adult
```

---

## If Statement (False Condition)

```text
age: N = 15

if age >= 18 {
    print("Adult")
}
```

Output:

```text
(no output)
```

---

## If Else Statement

```text
age: N = 15

if age >= 18 {
    print("Adult")
}
else {
    print("Minor")
}
```

Output:

```text
Minor
```

---

## Else If Statement

```text
marks: N = 85

if marks >= 90 {
    print("A")
}
else if marks >= 80 {
    print("B")
}
else {
    print("C")
}
```

Output:

```text
B
```

---

## Nested Conditionals

```text
age: N = 20
citizen: B = true

if age >= 18 {

    if citizen {
        print("Eligible")
    }

}
```

Output:

```text
Eligible
```

---

## Logical Conditions

Conditions may use logical operators.

```text
if age >= 18 && citizen {
    print("Eligible")
}
```

```text
if isAdmin || isModerator {
    print("Access Granted")
}
```

```text
if !isBlocked {
    print("Allowed")
}
```

---

## Ternary Operator

The ternary operator provides a concise way to choose between two values based on a condition.

Syntax:

```text
condition ? value_if_true : value_if_false
```

Example:

```text
age: N = 20

status: S = age >= 18 ? "Adult" : "Minor"

print(status)
```

Output:

```text
Adult
```

---

Example:

```text
score: N = 45

result: S = score >= 50 ? "Pass" : "Fail"

print(result)
```

Output:

```text
Fail
```

---

Ternary expressions may be used anywhere an expression is allowed.

```text
print(isAdmin ? "Admin" : "User")
```

```text
largest: N = x > y ? x : y
```

---

## Multiline Expressions

Expressions may span multiple lines when enclosed in parentheses.

Within a parenthesized expression, newline characters are ignored.

Valid:

```text
status: S = (
    age >= 18
        ? "Adult"
        : "Minor"
)
```

```text
result: S = (
    isAdmin
        ? "Admin"
        : citizen
            ? "Citizen"
            : "Guest"
)
```

```text
print(
    age >= 18
        ? (
            citizen
                ? "Eligible"
                : "Not Citizen"
        )
        : "Too Young"
)
```

```text
value: N = (
    10 +
    20 *
    3
)
```

```text
isEligible: B = (
    age >= 18 &&
    citizen
)
```

---

Outside of parentheses, expressions must remain on a single line.

Invalid:

```text
status: S = age >= 18
    ? "Adult"
    : "Minor"
```

Error:

```text
Unexpected token QUESTION.
```

---

## Parenthesized Expressions

Parentheses may be used to group expressions and improve readability.

```text
value: N = (
    10 + 20
) * 3
```

```text
result: S = (
    isAdmin
        ? "Admin"
        : "User"
)
```

Parentheses do not create a new scope and do not affect the resulting value other than controlling expression grouping.

---

## Ternary Compatibility Rules

Both branches of a ternary expression must produce compatible values.

Valid:

```text
message: S = isAdmin ? "Admin" : "User"
```

```text
value: N = x > y ? x : y
```

Invalid:

```text
value: N = condition ? 10 : "Hello"
```

Error:

```text
Ternary branches must produce compatible types.
```

---

## Scope Rules

Variables declared inside a conditional block only exist inside that block.

```text
if true {
    message: S = "Hello"
}

print(message)
```

Error:

```text
Variable 'message' does not exist.
```

---

Variables outside the block remain accessible inside the block.

```text
name: S = "Varun"

if true {
    print(name)
}
```

Output:

```text
Varun
```

---

## Variable Shadowing

Local variables may shadow outer variables.

```text
name: S = "Global"

if true {
    name: S = "Local"

    print(name)
}
```

Output:

```text
Local
```

```text
print(name)
```

Output:

```text
Global
```

---

## Valid Condition Types

Conditions must evaluate to a boolean value.

Valid:

```text
if age >= 18 {
}
```

```text
if isAdmin {
}
```

```text
if score > 50 && passed {
}
```

Invalid:

```text
if 10 {
}
```

Error:

```text
Condition must evaluate to a Boolean value.
```
