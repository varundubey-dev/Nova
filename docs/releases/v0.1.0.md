# NOVA v0.1.0

The first public release of NOVA.

NOVA v0.1 establishes the core execution pipeline of the language:

```
Source Code
    ↓
Lexer
    ↓
Parser
    ↓
Abstract Syntax Tree (AST)
    ↓
Interpreter
```

This release provides the foundation for every future version of NOVA by introducing primitive datatypes, variable declarations, arithmetic evaluation, console output, runtime validation, and the command-line interface.

---

## Highlights

- Initial lexer implementation
- Recursive-descent parser
- Abstract Syntax Tree (AST)
- Tree-walk interpreter
- Runtime environments
- Variable declarations
- Variable assignment
- Numbers (`N`)
- Strings (`S`)
- Arithmetic expressions
- `print()` output
- Single-line comments
- Multi-line comments
- Runtime datatype validation
- Uninitialized variable detection
- Command-line interface (`nova`)

## Next Release

Planned for v0.2:

- Booleans (`B`)
- Any Type (`U`)
- Constants
- Null values
- Comparison operators
- Logical operators

These additions will introduce NOVA's first complete primitive type system and enable conditional expressions and logical evaluation.
