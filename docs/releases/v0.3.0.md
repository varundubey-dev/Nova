# NOVA v0.3.0

The third public release of NOVA.

NOVA v0.3 introduces arrays, NOVA's first collection type, along with support for typed arrays, nested arrays, array indexing, and array mutation.

This release also introduces source-aware diagnostics, allowing parser, type, and runtime errors to precisely identify their location within user source code.

---

## Highlights

* Arrays
* Typed arrays
* Multi-type arrays
* Nested arrays
* Array indexing
* Array mutation
* Immutable array enforcement
* Source-aware diagnostics
* Improved runtime validation

## Next Release

Planned for v0.4:

* Maps (`M`)
* Schema declarations
* Map literals
* Property access
* Property mutation
* Optional properties
* Nested schemas
* Arrays of schemas

These additions will introduce structured data through reusable schemas and strongly typed map instances, enabling NOVA to model complex real-world objects while preserving runtime type safety.
