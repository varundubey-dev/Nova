# NOVA v0.2.0

The second public release of NOVA.

NOVA v0.2 extends the language with a complete primitive type system by introducing booleans, the any type, constants, null values, comparison operators, and logical operators.

These additions significantly increase the expressiveness of the language while preserving NOVA's simple execution model and runtime type safety.

---

## Highlights

* Booleans (`B`)
* Any Type (`U`)
* Constants (`::`)
* Deferred constant initialization
* Null values
* Comparison operators
* Equality operators
* Logical operators
* Runtime datatype validation
* Immutable constant enforcement
* Improved interpreter runtime
